# UI Skill System for Claude Code

A complete design system for Claude Code with 5 curated UI styles and 3 utility modules.

## Installation

```bash
/plugin marketplace add Viabled-Edge/claude-code-ui-skills
/plugin install soft-ui-system@claude-code-ui-skills
```

## Available Skills

### UI Styles
- **soft-ui-system** - Neumorphic, rounded, gentle interfaces
- **sharp-ui** - Brutalist, high-contrast, angular designs
- **glass-ui-system** - Glassmorphism, frosted, luminous effects
- **glossy-ui-system** - Premium, reflective, polished surfaces
- **retro-ui-system** - Synthwave, CRT, neon aesthetics

### Utility Modules
- **layout-system** - Panel-based layouts, scroll containment
- **form-patterns** - Form layouts and validation UX
- **motion-system** - Animation timing and micro-interactions

## Usage

After installing a skill, Claude Code will automatically apply the design system when you ask it to build UI components.
